Цель атаки Исчерпать пропускную способность канала или ресурсы сетевого оборудования Исчерпать ресурсы самого приложения (процессорное время, память, соединения с базой данных, логику обработки).
Тип атак Объемные атаки (отправка огромного количества бессмысленных пакетов):
  • UDP Flood
  • ICMP Flood
  • DNS/NTP/Memcached Amplification
Протокольные атаки (эксплуатация слабостей сетевых протоколов):
  • TCP SYN Flood
  • ACK/PUSH ACK Flood
  • Fragmented Packet Attacks
  • HTTP/HTTPS Flood
  • Slow-Rate Attacks
  • DNS Query Flood
  • Атаки на API
  • Инъекции и эксплойты
  • Directory Traversal / Path Manipulation
  • Session Hijacking / Cookie Tampering
Пропускная способность атаки Мощность атаки ограничена скоростью 300 Гбит/с. При больших значениях трафик, превышающий 300 Гбит/с, отбрасывается до классификации на легитимный и паразитный.
  • Амплифицирующие атаки с неизменными L4 параметрами: до 1.5 Тбит/с на территории РФ, до 8 Тбит/с суммарно.
  • Statefull атаки: без кратковременной деградации: 500 Гбит/с кратковременной деградацией (до 30 минут) до 800 Гбит/с на территории РФ, и до 3 Тбит/с суммарно.
Механизмы фильтрации ACL, BGP blackhole, фильтрация по IP/ASN, rate limiting, L4-фильтры WAF, антибот-защита, JS-челленджи, поведенческий анализ, CAPTCHA
Время реакции Не более 100 мс Не более 30 мс.
Вносимая задержка системой фильтрации - не более 3 мс
