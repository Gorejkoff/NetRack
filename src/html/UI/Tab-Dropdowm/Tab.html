У каждого клиента есть лицевой счёт, на котором хранится остаток денежных средств. Со счёта ежедневно списывается оплата за аренду виртуального сервера. Пополнить счёт можно через раздел «Финансы» — «Платежи».

Например, вы используете VDS-Форсаж за 749 рублей в месяц. В феврале (28 дней) со счёта будет списываться сумма 749 / 28 = 26,75 рублей ежедневно.

Таким образом тарифицируются виртуальные серверы и дополнительные услуги: панель управления, дополнительные IP-адреса, дополнительные ресурсы сервера, лицензии Windows Server.

Пока на счёте имеются деньги — VPS и остальные услуги работают. Посмотреть список всех расходов можно в Личном кабинете в разделе «Финансы» — «Расходы».
